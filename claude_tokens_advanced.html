🔬 토큰이란 무엇인가?

토큰은 모델이 처리하는 텍스트의 최소 단위.
영어 기준 1토큰 ≈ 4글자 ≈ 0.75단어
언어·내용에 따라 비율이 달라짐.
실제 변환 예시
"Hello World" 2~3 tokens
1,000 영어 단어 ~750 tokens
일반 웹페이지 (10kB) ~2,500 tokens
연구 논문 PDF (500kB) ~125,000 tokens

(Opus 4.7, Opus 4.6, Sonnet 4.6 기준)

🗂️ 토큰의 6가지 종류

① Base Input
기본 입력 토큰
사용자가 보내는 프롬프트, 시스템 프롬프트, 이전 대화 내역 전체
기준 가격 (1×)
② Output
출력 토큰
모델이 생성하는 응답 텍스트. 입력보다 비싼 경우가 많음 (최대 5×)
3~5× 입력 단가
③ Cache Write
캐시 쓰기 토큰
프롬프트 캐싱 시 처음 저장할 때 소비. 5분/1시간 두 종류 존재
1.25× (5분) / 2× (1시간)
④ Cache Read
캐시 읽기 토큰
캐시된 내용을 재사용할 때 소비. 매우 저렴 — 비용 절감의 핵심
0.1× 입력 단가 (90% 할인)
⑤ Thinking
thinking 토큰
Extended Thinking / Adaptive Thinking 시 내부 추론에 사용. 출력 토큰으로 과금
출력 토큰 단가 동일
⑥ Tool System
도구 시스템 토큰
Tool Use 활성화 시 자동 추가되는 시스템 프롬프트 토큰 (모델별 346 등)
입력 토큰으로 과금

🧮 프롬프트 캐싱 — 수학으로 이해하기

// 5분 캐시: 몇 번 읽어야 이득?
쓰기 비용 = 1.25 × 입력 단가
읽기 비용 = 0.10 × 입력 단가
→ 1회 읽기만으로 본전 (1.25 < 1.0 + 0.10)

// 1시간 캐시: 몇 번 읽어야 이득?
쓰기 비용 = 2.0 × 입력 단가
→ 2회 읽기부터 이득 (2.0 < 1.0×2 + 0.10×2)
실제 절감 예시 (Sonnet 4.6 기준)
50k 토큰 시스템 프롬프트를 10회 반복 요청

캐싱 없음: 50,000 × 10 × $3 / 1M = $1.50
5m 캐싱: 50,000×$3.75 + 50,000×9×$0.30 / 1M = $0.322
절감율: ~78%
캐싱 방법 2가지
• cache_control 최상위 설정 → 자동 관리 (권장)
• 개별 content block에 설정 → 세밀한 제어
캐시 할인은 Batch API 50% 할인과 중첩 적용 가능

🧠 Thinking 토큰 과금

Thinking 토큰은 출력 토큰으로 과금.
Claude 4.x에서는 Extended Thinking 대신 Adaptive Thinking 사용.
기본 예산 = 수만 토큰 (모델별 상이)
// Sonnet 4.6 기준 1회 요청 예시
Input: 5,000 tokens × $3 = $0.015
Thinking: 20,000 tokens × $15 = $0.300
Output: 1,000 tokens × $15 = $0.015
Total = $0.330 (thinking이 91%!)
Thinking 비용 제어 방법
  • 1
    /effort low — 노력 수준 낮춤 → thinking 예산 축소
  • 2
    MAX_THINKING_TOKENS=8000 환경변수로 예산 하드 캡
  • 3
    /config에서 thinking 완전 비활성화
  • ✓
    복잡한 아키텍처/멀티스텝 추론에서는 thinking이 오히려 재작업 비용 절감
Thinking 토큰은 캐시 불가 — 매 요청마다 새로 소비됨

🤖 Claude Managed Agents 과금

Managed Agents는 토큰 + 세션 런타임 2가지로 과금.
Batch API, Fast Mode, 데이터 레지던시 할증은 적용 불가.
세션 런타임 요금
$0.08 / 시간
상태가 running인 동안만 측정.
idle, rescheduling, terminated 상태는 미포함.
실제 계산 예시 (Opus 4.7, 1시간 코딩 세션)
// 캐싱 없는 경우
Input 50k × $5/M = $0.250
Output 15k × $25/M = $0.375
Runtime 1h × $0.08 = $0.080
합계 = $0.705
// 40k 토큰이 캐시 읽기인 경우
Uncached 10k × $5/M = $0.050
CacheRead 40k × $0.5/M = $0.020
Output 15k × $25/M = $0.375
Runtime 1h × $0.08 = $0.080
합계 = $0.525 (25% 절감)
